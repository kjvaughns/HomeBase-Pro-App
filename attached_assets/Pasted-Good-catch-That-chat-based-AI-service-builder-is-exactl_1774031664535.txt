## 🔧 Prompt: Replace AI Service Builder with Smart Service Setup

## Phase 1: Remove the Current System

Identify and remove:

* chat-based service builder UI
* unnecessary conversational flows
* anything that slows down creating a service
* any AI that requires typing prompts to function

If AI adds friction, remove it.

---

## Phase 2: Build Simple Service Creation

Create a clean, fast “Add Service” flow.

### Required Fields:

* Service Name

* Description (optional but recommended)

* Pricing Type:

  * Fixed price
  * Starting price
  * Quote required

* Price (if applicable)

* Duration

* Service Area (optional if global)

---

## Phase 3: Smart Pricing Models (Important)

Support real world service logic:

### 1. Flat Rate

Example: Lawn cut = $50

### 2. Size Based Pricing

Example:

* Small yard (0–5,000 sqft)
* Medium yard (5,000–10,000 sqft)
* Large yard (10,000+ sqft)

### 3. Custom Quote

User submits request, provider confirms price

UI must make this simple, not overwhelming.

---

## Phase 4: AI Enhancement (Non-Chat, Inline Only)

AI should assist silently and instantly.

### Examples:

When typing service name:

* Suggest common services
* Auto-complete categories

When adding description:

* “Improve description” button
* Auto rewrite to sound more professional

When setting price:

* Suggest price ranges based on service type + location

When finishing service:

* Tip:
  “Providers with photos get 30% more bookings”
  “Adding 2 more services increases visibility”

When leaving fields blank:

* Suggest smart defaults

---

## Phase 5: AI Suggestion System (Global Behavior)

Replace chat AI with:

* Inline suggestions
* Small tips
* Smart autofill
* One-click improvements

Design:

* subtle UI (not intrusive)
* small hint text or suggestion cards

NO chat bubbles
NO typing prompts
NO “ask AI” flows unless absolutely needed

* Service creation must take under 60 seconds
* No step should feel confusing
