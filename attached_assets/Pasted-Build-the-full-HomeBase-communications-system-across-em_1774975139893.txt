Build the full HomeBase communications system across email, push notifications, in app notifications, and provider to client messaging.

Main goal:
Create a complete HomeBase branded communications system that handles both platform level communications and service provider to client communications. This system must feel premium, trustworthy, and operationally useful. It should support onboarding, account creation, reminders, booking updates, job updates, invoice updates, payments, and provider sent client communications. Resend is already being used for email. Design this as a real production ready communications layer, not a basic email template system.

Primary goals:
1. HomeBase sends branded lifecycle emails and notifications for all major platform events
2. Providers can send emails and text updates to their clients through HomeBase
3. Clients and providers receive useful reminders, status updates, and confirmations
4. Notifications stay in sync across email, push, in app, and SMS where supported
5. Everything is tied to real booking, schedule, invoice, payment, and account data

Build this feature end to end.
